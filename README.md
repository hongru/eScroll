eScroll
=======

Easy Scroller for mobile web app

Usage
====
<pre>
var scroller = new EScroll({
    element: '#wrapper',
    isEase: true, // |false
    lockX: true,
    lockY: false
});

scroller.scrollTo(100);
scroller.setActiveEl('#con1');
</pre>
